Ejercicio 1: Números Pares e Impares

Escribe un programa que solicite al usuario un número entero positivo y luego imprima todos los números pares e impares desde 1 hasta el número ingresado.

// Ejercicio 1: Números Pares e Impares
#include <iostream>

int main() {
    int numero;
    std::cout << "Ingrese un número entero positivo: ";
    std::cin >> numero;

    std::cout << "Números pares hasta " << numero << ": ";
    for (int i = 1; i <= numero; ++i) {
        if (i % 2 == 0) {
            std::cout << i << " ";
        }
    }
    std::cout << std::endl;

    std::cout << "Números impares hasta " << numero << ": ";
    for (int i = 1; i <= numero; ++i) {
        if (i % 2 != 0) {
            std::cout << i << " ";
        }
    }
    std::cout << std::endl;

    return 0;
}

Ejercicio 2: Suma de Números

Escribe un programa que solicite al usuario números enteros positivos y calcule la suma total. El programa debe detenerse cuando el usuario ingrese un número negativo y mostrar la suma de los números ingresados hasta ese momento.

// Ejercicio 2: Suma de Números
#include <iostream>

int main() {
    int numero, suma = 0;
    std::cout << "Ingrese números enteros positivos (ingrese un número negativo para terminar): ";
    while (true) {
        std::cin >> numero;
        if (numero < 0) {
            break;
        }
        suma += numero;
    }
    std::cout << "La suma de los números ingresados es: " << suma << std::endl;

    return 0;
}

Ejercicio 3: Números Primos

Escribe un programa que solicite al usuario un número entero positivo y determine si es un número primo o no. Un número primo es aquel que es divisible solo por 1 y por sí mismo.

// Ejercicio 3: Números Primos
#include <iostream>

int main() {
    int numero;
    bool esPrimo = true;

    std::cout << "Ingrese un número entero positivo: ";
    std::cin >> numero;

    if (numero <= 1) {
        esPrimo = false;
    } else {
        for (int i = 2; i <= numero / 2; ++i) {
            if (numero % i == 0) {
                esPrimo = false;
                break;
            }
        }
    }

    if (esPrimo) {
        std::cout << numero << " es un número primo." << std::endl;
    } else {
        std::cout << numero << " no es un número primo." << std::endl;
    }

    return 0;
}

Ejercicio 4: Números Fibonacci

Escribe un programa que genere y muestre los primeros N números de la secuencia de Fibonacci, donde N es un número entero ingresado por el usuario.

// Ejercicio 4: Números Fibonacci
#include <iostream>

int main() {
    int n, t1 = 0, t2 = 1, siguiente;

    std::cout << "Ingrese el número de términos de la secuencia de Fibonacci: ";
    std::cin >> n;

    std::cout << "Secuencia de Fibonacci de " << n << " términos: ";

    for (int i = 1; i <= n; ++i) {
        std::cout << t1 << ", ";
        siguiente = t1 + t2;
        t1 = t2;
        t2 = siguiente;
    }

    return 0;
}

Ejercicio 5: Tabla de Multiplicar

Escribe un programa que solicite al usuario un número entero positivo y muestre la tabla de multiplicar de ese número hasta el 10.

// Ejercicio 5: Tabla de Multiplicar
#include <iostream>

int main() {
    int numero;

    std::cout << "Ingrese un número entero positivo: ";
    std::cin >> numero;

    std::cout << "Tabla de multiplicar del " << numero << ":" << std::endl;
    for (int i = 1; i <= 10; ++i) {
        std::cout << numero << " x " << i << " = " << numero * i << std::endl;
    }

    return 0;
}

Ejercicio 6: Suma de Números Pares

Escribe un programa que calcule la suma de los primeros N números pares. El valor de N es ingresado por el usuario.

// Ejercicio 6: Suma de Números Pares
#include <iostream>

int main() {
    int n, suma = 0;

    std::cout << "Ingrese el número de términos pares a sumar: ";
    std::cin >> n;

    for (int i = 1; i <= n; ++i) {
        suma += (2 * i);
    }

    std::cout << "La suma de los primeros " << n << " números pares es: " << suma << std::endl;

    return 0;
}
