Retorno de Valores en C++

Introducción

En C++, el retorno de valores se refiere a devolver un valor específico de una función al lugar donde se llamó. Este valor puede ser de cualquier tipo de datos, como int, float, double, etc.

Retorno de Valores en Funciones

Para devolver un valor desde una función, se utiliza la palabra clave return, seguida del valor que se desea devolver. El tipo de dato del valor devuelto debe coincidir con el tipo de dato especificado en la declaración de la función.

// Declaración de la función
int cuadrado(int x);

// Definición de la función
int cuadrado(int x) {
    return x * x;
}

Uso del Valor Retornado

El valor retornado por una función puede ser asignado a una variable, utilizado en una expresión o simplemente ignorado, dependiendo de los requisitos del programa.

// Llamada a la función y uso del valor retornado
int resultado = cuadrado(5); // El valor retornado por la función se asigna a la variable 'resultado'
std::cout << "El cuadrado de 5 es: " << resultado << std::endl;

Ejemplo

A continuación se muestra un ejemplo completo que utiliza el retorno de valores en una función para calcular el área de un círculo.

// Declaración de la función
double calcularAreaCirculo(double radio);

int main() {
    double radio = 5.0;
    double area = calcularAreaCirculo(radio);
    std::cout << "El área del círculo con radio " << radio << " es: " << area << std::endl;
    return 0;
}

// Definición de la función
double calcularAreaCirculo(double radio) {
    return 3.14159 * radio * radio;
}
