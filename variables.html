Módulo 2: Variables y Tipos de Datos

1. Variables y constantes

En C++, una variable es un espacio de almacenamiento nombrado que puede contener un valor. Las variables pueden tener diferentes tipos de datos, como enteros, flotantes, caracteres, booleanos, entre otros. También exploraremos el concepto de constantes, que son valores que no cambian durante la ejecución del programa.

2. Tipos de datos básicos

En este apartado, analizaremos los tipos de datos básicos disponibles en C++, incluyendo:

  • Enteros: Representan números enteros, como int, short, long, etc.
  • Flotantes: Representan números con parte decimal, como float y double.
  • Caracteres: Representan caracteres individuales, como char.
  • Booleanos: Representan valores verdadero o falso, como bool.

Explicaremos cómo declarar variables de estos tipos y cómo asignarles valores.

3. Declaración, asignación y uso de variables

En esta sección, aprenderemos cómo declarar variables en C++, asignarles valores y utilizarlas en expresiones y operaciones. También discutiremos las convenciones de nomenclatura para nombrar variables de manera significativa y descriptiva.

Ejemplo:

// Declaración y asignación de una variable
int edad = 25;

// Uso de la variable en una operación
int edadDespuesDeDiezAnios = edad + 10;

// Impresión del resultado
std::cout << "Edad después de diez años: " << edadDespuesDeDiezAnios << std::endl;

4. Conversión de tipos de datos

A veces necesitamos convertir un tipo de dato en otro. Por ejemplo, convertir un número entero en un número de punto flotante. Exploraremos los diferentes métodos de conversión de tipos de datos en C++ y cómo evitar la pérdida de precisión o la truncación de datos.

Ejemplo:

// Conversión de tipos de datos
int numeroEntero = 10;
double numeroFlotante = static_cast<double>(numeroEntero);
std::cout << "Número flotante: " << numeroFlotante << std::endl;
