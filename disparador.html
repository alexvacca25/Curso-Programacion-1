Uso de break y continue en C++

Descripción

break y continue son palabras clave utilizadas en bucles en C++ para controlar el flujo de ejecución.

Uso de break

break se utiliza para salir de un bucle de manera prematura, terminando inmediatamente la ejecución del bucle.

// Ejemplo de uso de break
for (int i = 0; i < 10; ++i) {
    if (i == 5) {
        break; // Sale del bucle cuando i es igual a 5
    }
    std::cout << i << std::endl;
}

Uso de continue

continue se utiliza para omitir el resto del código en una iteración del bucle y pasar a la siguiente iteración.

// Ejemplo de uso de continue
for (int i = 0; i < 10; ++i) {
    if (i % 2 == 0) {
        continue; // Omite la impresión de números pares
    }
    std::cout << i << std::endl;
}
