Declaración y Definición de Funciones en C++

Introducción

Las funciones en C++ son bloques de código que realizan una tarea específica. Pueden ser declaradas y definidas en diferentes partes de un programa para mejorar la modularidad y la legibilidad del código.

Declaración de Funciones

La declaración de una función informa al compilador sobre la existencia de una función antes de que se llame. La declaración de una función incluye el tipo de retorno de la función, el nombre de la función y los tipos de sus parámetros, pero no incluye el cuerpo de la función.

// Declaración de una función
int suma(int a, int b); // Declara una función llamada 'suma' que toma dos parámetros enteros y devuelve un entero

Definición de Funciones

La definición de una función proporciona la implementación real de la función, incluyendo el cuerpo de la función.

// Definición de la función 'suma'
int suma(int a, int b) {
    return a + b;
}

Ejemplo

A continuación, se muestra un ejemplo completo que utiliza la declaración y definición de una función para calcular la suma de dos números.

// Declaración de la función
int suma(int a, int b);

int main() {
    int resultado = suma(5, 3);
    std::cout << "La suma es: " << resultado << std::endl;
    return 0;
}

// Definición de la función
int suma(int a, int b) {
    return a + b;
}
