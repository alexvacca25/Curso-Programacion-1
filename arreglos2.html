Punteros y Gestión de Memoria Dinámica en C++

Introducción

Los punteros son variables que almacenan direcciones de memoria. La gestión de memoria dinámica en C++ permite asignar y liberar memoria en tiempo de ejecución.

Punteros

Un puntero es una variable que almacena la dirección de memoria de otra variable. Se declara utilizando el operador asterisco (*) antes del nombre de la variable.

// Declaración de un puntero
int *ptr;

Asignación de Memoria Dinámica

La asignación de memoria dinámica se realiza utilizando el operador new. Devuelve un puntero al tipo de dato especificado.

// Asignación de memoria dinámica para un entero
int *ptr = new int;

Liberación de Memoria Dinámica

La liberación de memoria dinámica se realiza utilizando el operador delete. Libera la memoria asignada previamente por new.

// Liberación de memoria dinámica
delete ptr;
