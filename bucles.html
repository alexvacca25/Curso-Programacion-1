Bucles en C++

Descripción

Los bucles en C++ son estructuras de control que permiten repetir un bloque de código varias veces. Los tres bucles más comunes son: for, while y do-while.

Bucle for

El bucle for se utiliza cuando se conoce de antemano el número de iteraciones que se deben realizar.

// Ejemplo de bucle for
for (int i = 0; i < 5; ++i) {
    std::cout << "Iteración " << i + 1 << std::endl;
}

Bucle while

El bucle while se utiliza cuando la condición de terminación no es conocida de antemano y el bloque de código debe ejecutarse mientras la condición sea verdadera.

// Ejemplo de bucle while
int contador = 0;
while (contador < 5) {
    std::cout << "Iteración " << contador + 1 << std::endl;
    ++contador;
}

Bucle do-while

El bucle do-while es similar al bucle while, pero la condición se evalúa al final del bloque de código, lo que garantiza que el bloque se ejecute al menos una vez.

// Ejemplo de bucle do-while
int contador = 0;
do {
    std::cout << "Iteración " << contador + 1 << std::endl;
    ++contador;
} while (contador < 5);
