Parámetros y Argumentos en C++

Introducción

En C++, los parámetros son variables que se utilizan para pasar información a una función. Los argumentos son los valores reales que se pasan a esos parámetros cuando se llama a la función.

Parámetros de Funciones

Los parámetros de una función se definen entre paréntesis en la declaración y definición de la función. Estos parámetros actúan como variables locales dentro de la función.

// Declaración de la función con parámetros
void saludar(std::string nombre);

// Definición de la función con parámetros
void saludar(std::string nombre) {
    std::cout << "¡Hola, " << nombre << "!" << std::endl;
}

Argumentos de Funciones

Los argumentos de una función son los valores reales que se pasan a los parámetros cuando se llama a la función. Estos valores pueden ser constantes, variables o expresiones.

// Llamada a la función con argumentos
saludar("Juan"); // "Juan" es el argumento que se pasa al parámetro 'nombre'

Ejemplo

A continuación se muestra un ejemplo completo que utiliza parámetros y argumentos en una función para sumar dos números.

// Declaración de la función
int sumar(int a, int b);

int main() {
    int resultado = sumar(5, 3); // Llamada a la función con argumentos
    std::cout << "La suma es: " << resultado << std::endl;
    return 0;
}

// Definición de la función
int sumar(int a, int b) {
    return a + b;
}
