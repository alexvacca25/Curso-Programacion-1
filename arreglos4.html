Vectores y Listas en C++

Introducción

Los vectores y las listas son estructuras de datos que permiten almacenar colecciones de elementos en C++. Los vectores son arreglos dinámicos, mientras que las listas son contenedores enlazados.

Vectores

Un vector es una secuencia ordenada de elementos del mismo tipo, que se almacenan en una memoria contigua.

// Declaración e inicialización de un vector
#include <vector>

std::vector<int> numeros = {1, 2, 3, 4, 5};

Listas

Una lista es una colección de elementos, donde cada elemento está vinculado al siguiente mediante punteros.

// Declaración e inicialización de una lista
#include <list>

std::list<int> numeros = {1, 2, 3, 4, 5};

Acceso a Elementos

Los elementos en un vector y en una lista se pueden acceder utilizando el operador de corchetes [] (para vectores) o iteradores (para listas).

// Acceso a elementos de un vector
int elemento = numeros[2]; // Accede al tercer elemento del vector 'numeros'
// Acceso a elementos de una lista utilizando iteradores
std::list<int>::iterator it = numeros.begin();
std::advance(it, 2); // Avanza el iterador dos posiciones
int elemento = *it;
