Cadenas de Caracteres (Strings) en C++

Introducción

Las cadenas de caracteres en C++ se pueden representar utilizando la clase std::string del estándar de la biblioteca estándar de C++. Esta clase proporciona una forma conveniente de manipular cadenas de caracteres.

Declaración e Inicialización

Las cadenas de caracteres se pueden declarar e inicializar de varias formas en C++.

// Declaración e inicialización de una cadena vacía
std::string cadenaVacia;

// Declaración e inicialización de una cadena con un valor inicial
std::string saludo = "¡Hola, mundo!";

Acceso a Caracteres

Los caracteres individuales en una cadena de caracteres se pueden acceder utilizando el operador de corchetes [] junto con un índice.

// Acceso a caracteres individuales
char primerCaracter = saludo[0]; // Accede al primer carácter de la cadena 'saludo'

Operaciones Comunes

La clase std::string proporciona numerosas funciones y operadores para realizar operaciones comunes en cadenas de caracteres, como concatenación, búsqueda, inserción, eliminación, etc.

// Concatenación de cadenas
std::string nombre = "Juan";
std::string apellido = "Pérez";
std::string nombreCompleto = nombre + " " + apellido;
