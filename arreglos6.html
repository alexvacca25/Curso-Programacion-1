Ejercicios Propuestos sobre Vectores y Listas en C++

Ejercicio 1: Buscar un Elemento

Escribe un programa que solicite al usuario un número entero y luego busque si ese número está presente en un vector predefinido de enteros.

Ejercicio 2: Fusionar Listas

Escribe un programa que fusiona dos listas de números enteros ordenadas en una sola lista ordenada. El programa debe asegurarse de que la lista resultante esté también ordenada.

Ejercicio 3: Eliminar Repetidos

Escribe un programa que elimine los elementos duplicados de un vector o una lista de números enteros.

Ejercicio 4: Suma de Elementos Pares

Escribe un programa que calcule la suma de todos los elementos pares en un vector o una lista de números enteros.

Ejercicio 5: Rotación de Elementos

Escribe un programa que rote los elementos de un vector o una lista a la izquierda o a la derecha según la cantidad de posiciones especificadas por el usuario.
