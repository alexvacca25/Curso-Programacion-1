Ejercicios Prácticos de Funciones en C++

Ejercicio 1: Función para Calcular el Máximo

Escribe una función en C++ llamada maximo que tome dos números como argumentos y devuelva el mayor de los dos.

// Ejercicio 1: Función para Calcular el Máximo
#include <iostream>

int maximo(int a, int b) {
    return (a > b) ? a : b;
}

int main() {
    int num1, num2;

    std::cout << "Ingrese el primer número: ";
    std::cin >> num1;

    std::cout << "Ingrese el segundo número: ";
    std::cin >> num2;

    std::cout << "El máximo es: " << maximo(num1, num2) << std::endl;

    return 0;
}

Ejercicio 2: Función para Calcular el Factorial

Escribe una función en C++ llamada factorial que tome un número entero como argumento y devuelva su factorial.

// Ejercicio 2: Función para Calcular el Factorial
#include <iostream>

int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

int main() {
    int num;

    std::cout << "Ingrese un número: ";
    std::cin >> num;

    std::cout << "El factorial de " << num << " es: " << factorial(num) << std::endl;

    return 0;
}

Ejercicio 3: Función para Calcular la Potencia

Escribe una función en C++ llamada potencia que tome dos números enteros como argumentos y devuelva el primero elevado a la potencia del segundo.

// Ejercicio 3: Función para Calcular la Potencia
#include <iostream>
#include <cmath>

double potencia(double base, int exponente) {
    return std::pow(base, exponente);
}

int main() {
    double base;
    int exponente;

    std::cout << "Ingrese la base: ";
    std::cin >> base;

    std::cout << "Ingrese el exponente: ";
    std::cin >> exponente;

    std::cout << "El resultado de " << base << " elevado a la potencia " << exponente << " es: " << potencia(base, exponente) << std::endl;

    return 0;
}
