Creación de una cuenta en Replit

Para comenzar a programar en C++ en Replit, primero necesitas crear una cuenta en la plataforma. Sigue estos pasos:

  1. Visita el sitio web de Replit: replit.com.
  2. Haz clic en "Sign Up" o "Registrarse" para crear una cuenta nueva.
  3. Rellena el formulario de registro con tu nombre de usuario, dirección de correo electrónico y contraseña.
  4. Una vez que hayas completado el registro, ¡ya estás listo para empezar!

Ejemplo Básico de C++

Aquí tienes un ejemplo básico de un programa en C++ que imprime "Hola Mundo":

// Programa en C++ que imprime "Hola Mundo"
#include <iostream>

int main() {
    std::cout << "Hola Mundo" << std::endl;
    return 0;
}

Este programa utiliza la biblioteca estándar de C++ para imprimir "Hola Mundo" en la consola.
