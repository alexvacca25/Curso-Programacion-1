Paso de Parámetros en C++

Introducción

En C++, los parámetros de funciones pueden ser pasados por valor o por referencia. El paso por valor implica que una copia del valor del parámetro se pasa a la función, mientras que el paso por referencia implica pasar una referencia a la variable original.

Paso de Parámetros por Valor

Al pasar un parámetro por valor, se crea una copia del valor original dentro de la función. Cualquier modificación realizada en el parámetro dentro de la función no afectará al valor original fuera de ella.

// Función con parámetro por valor
void duplicar(int x) {
    x = x * 2;
}

Paso de Parámetros por Referencia

Al pasar un parámetro por referencia, se pasa una referencia (dirección de memoria) al valor original. Cualquier modificación realizada en el parámetro dentro de la función afectará al valor original fuera de ella.

// Función con parámetro por referencia
void duplicarPorReferencia(int & x) {
    x = x * 2;
}

Ejemplo

A continuación se muestra un ejemplo completo que utiliza tanto el paso de parámetros por valor como por referencia en una función.

// Función con parámetro por valor
void duplicar(int x) {
    x = x * 2;
}

// Función con parámetro por referencia
void duplicarPorReferencia(int & x) {
    x = x * 2;
}

int main() {
    int num = 5;
    
    // Paso por valor
    duplicar(num);
    std::cout << "Duplicar por valor: " << num << std::endl; // Imprime 5

    // Paso por referencia
    duplicarPorReferencia(num);
    std::cout << "Duplicar por referencia: " << num << std::endl; // Imprime 10

    return 0;
}
