Arreglos en C++

Introducción

Los arreglos en C++ son estructuras de datos que permiten almacenar múltiples valores del mismo tipo bajo un solo nombre. Pueden ser unidimensionales (vectores) o multidimensionales (matrices).

Arreglos Unidimensionales

Un arreglo unidimensional es una lista de elementos del mismo tipo almacenados de manera contigua en la memoria.

// Declaración e inicialización de un arreglo unidimensional
int numeros[5] = {1, 2, 3, 4, 5};

Arreglos Multidimensionales

Un arreglo multidimensional es una colección de arreglos unidimensionales, donde cada arreglo unidimensional representa una fila o una columna de elementos.

// Declaración e inicialización de un arreglo bidimensional
int matriz[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

Acceso a Elementos

Para acceder a un elemento específico de un arreglo, se utiliza el nombre del arreglo seguido de un índice entre corchetes.

// Acceso a un elemento de un arreglo unidimensional
int valor = numeros[2]; // Accede al tercer elemento del arreglo 'numeros'
// Acceso a un elemento de un arreglo bidimensional
int elemento = matriz[1][2]; // Accede al elemento en la segunda fila y tercera columna de la matriz
