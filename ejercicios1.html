Ejercicios sobre el Uso de Variables y Constantes en C++

  1. Calculadora Simple: Escribir un programa que solicite al usuario dos números enteros y luego muestre la suma, resta, multiplicación y división de esos números.
  2. Conversión de Moneda: Escribir un programa que convierta una cantidad en dólares a euros.
  3. Área de un Círculo: Escribir un programa que calcule y muestre el área de un círculo.
  4. Calculadora de Índice de Masa Corporal (IMC): Escribir un programa que calcule el índice de masa corporal (IMC) de una persona.
  5. Cálculo de la Hipotenusa: Escribir un programa que calcule la longitud de la hipotenusa de un triángulo rectángulo.
  6. Promedio de Calificaciones: Escribir un programa que calcule el promedio de tres calificaciones.
  7. Cálculo del Perímetro y Área de un Rectángulo: Escribir un programa que calcule el perímetro y el área de un rectángulo.
  8. Tiempo de Viaje: Escribir un programa que calcule el tiempo de viaje necesario para recorrer una distancia a una velocidad constante.
  9. Calculadora de Interés Simple: Escribir un programa que calcule el interés simple sobre un préstamo.
  10. Cálculo de Volumen de un Cubo: Escribir un programa que calcule el volumen de un cubo.
