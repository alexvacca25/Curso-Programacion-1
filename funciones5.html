Funciones Recursivas en C++

Introducción

Una función recursiva es aquella que se llama a sí misma para resolver un problema. En C++, la recursión es una técnica poderosa que se utiliza para resolver problemas que pueden dividirse en subproblemas más pequeños.

Funciones Recursivas

Una función recursiva consta de dos partes: el caso base y el caso recursivo. El caso base es una condición que detiene la recursión y generalmente devuelve un valor constante o predefinido. El caso recursivo es donde se realiza la llamada recursiva a la misma función con un problema más pequeño.

// Ejemplo de función recursiva para calcular el factorial
int factorial(int n) {
    // Caso base: factorial de 0 es 1
    if (n == 0) {
        return 1;
    }
    // Caso recursivo: n * factorial(n-1)
    else {
        return n * factorial(n - 1);
    }
}

Ejemplo

A continuación se muestra un ejemplo completo que utiliza una función recursiva para calcular el factorial de un número.

// Definición de la función recursiva para calcular el factorial
int factorial(int n) {
    // Caso base: factorial de 0 es 1
    if (n == 0) {
        return 1;
    }
    // Caso recursivo: n * factorial(n-1)
    else {
        return n * factorial(n - 1);
    }
}

int main() {
    int num = 5;
    int fact = factorial(num);
    std::cout << "El factorial de " << num << " es: " << fact << std::endl;
    return 0;
}
